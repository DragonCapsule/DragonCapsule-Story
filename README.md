## 龙胶囊传奇

《DragonCapsule》具备开放式的平行宇宙多线故事情节，是由社群互动创作驱动的UGC故事, 《DragonCapsule》是DC创始团队讲述的第一个龙胶囊平行宇宙故事主线，欢迎对DC故事感兴趣的创作者加入_**DC-MPC**_（龙胶囊多维平行宇宙俱乐部）参与众创。
